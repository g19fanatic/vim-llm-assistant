==============================================================================
1. Introduction                        *llm-introduction*

vim-llm-assistant is a Vim plugin that integrates Large Language Models (LLMs)
directly into your Vim workflow. It allows you to interact with LLMs, save
responses in a dedicated buffer, and customize model selection.

==============================================================================
2. Commands                            *llm-commands*

:LLM [prompt]                          *:LLM*
    Process the current buffer and visible buffers with an LLM.
    If [prompt] is provided, it will be sent as a prompt to the LLM.

:SetLLMModel {model}                   *:SetLLMModel*
    Set the default LLM model to use. You can now take advantage of enhanced
    completion: when invoking this command (e.g. by typing :SetLLMModel and then
    pressing Ctrl‑D), Vim displays a list of available models fetched dynamically
    via the custom completion function.
    
:ListLLMModels                         *:ListLLMModels*
    List all available LLM models.

:SetLLMAdapter {adapter}               *:SetLLMAdapter*
    Set the default LLM adapter to use. Supports custom completion with
    Ctrl‑D to display available adapters.

:ListLLMAdapters                       *:ListLLMAdapters*
    List all registered LLM adapters.

==============================================================================
3. Configuration                       *llm-configuration*

The following variables can be set in your vimrc:

g:llm_default_model                    *g:llm_default_model*
    The default LLM model to use.
    Default: 'claude-3-7-sonnet-20250219'

g:llm_role                             *g:llm_role*
    The role to use with the LLM.
    Default: 'default-vim-role'

g:llm_default_adapter                  *g:llm_default_adapter*
    The default LLM adapter to use.
    Default: 'aichat'

g:llm_adapters                         *g:llm_adapters*
    A list of adapter names to load at startup. Each name corresponds to
    a file in autoload/llm/adapters/ (without the .vim extension).
    Default: ['aichat']

    Example:
        " Load multiple adapters
        let g:llm_adapters = ['aichat', 'custom_adapter']

Additional Note:
    The :SetLLMModel command now supports a dynamic completion feature. When
    entering the model name, simply press Ctrl‑D to view the list of models returned
    by the function llm#get_available_models(). This makes it easier to select the
    desired model without needing to type it out manually.

==============================================================================
4. Usage                               *llm-usage*

Basic usage:
    - Type `:LLM How do I improve this code?` to get suggestions.
    - Use `:SetLLMModel model_name` to change the default model.
    - Use `:ListLLMModels` to see available models.
    - Use `:SetLLMAdapter adapter_name` to change the default adapter.
    - Use `:ListLLMAdapters` to see available adapters.

Note: Support for visual selection will be added in a future version.
